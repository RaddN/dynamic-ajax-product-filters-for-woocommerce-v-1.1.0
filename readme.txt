== Installation ==

= From your WordPress dashboard: =

1. **Visit** Plugins > Add New
2. **Search** for "Dynamic AJAX Product Filters for WooCommerce"
3. **Activate** the plugin from your Plugins page
4. Configure filter settings in the WooCommerce settings panel under "Product Filters."

= Process of Using AJAX Product Filters for WooCommerce Store =

**Step 1: Configure Product Filters**
1. Navigate to **Admin** ‚Üí **Product Filters** ‚Üí **Form Manage**.
2. Enable the options for **Show Categories**, **Show Attributes**, **Show Tags**, **Show Price Range**, and **Show Rating** as needed.
3. Choose a filtering method in the use **URL-Based Filter** section:
   - With Query String (e.g., `?filters`)
   - With Permalinks
   - With Ajax

**Step 2: Add Products and Customize Filters**
1. Add WooCommerce products, categories, attributes, and tags.
2. Use the `[plugincy_filters]` shortcode to display filtering options wherever needed.
3. Customize the filter widget styles via **Admin** ‚Üí **Product Filters** ‚Üí **Form Style** tab.
